Given a string, find the length of the longest substring _T_ that contains at most _k_ distinct characters.

**Example 1:**
```
Input: s = "eceba", k = 2
Output: 3
Explanation: T is "ece" which its length is 3.
```
**Example 2:**
```
Input: s = "aa", k = 1
Output: 2
Explanation: T is "aa" which its length is 2.
```
